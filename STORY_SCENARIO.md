## 世界観（短編）

現実の街を歩くと、見えない「記憶の層」が浮かび上がる。
プレイヤーは青いスライムの勇者となり、街に漂うまものを退けながら、
自分で決めた「目的地」へ向かう旅を続ける。

## プレイヤー自作ストーリー（目的地システム）

1. **ストーリー** メニューを開く
2. タイトル・物語テキストを入力
3. **「いまの場所を目的地にする」** または 緯度経度を入力
4. マップ上を歩いて目的地半径 **40m** 以内に入るとイベント発動
5. 登録した物語テキストを表示し、達成フラグを保存

### 目的地データ構造（実装）

```json
{
  "id": "goal_001",
  "title": "かえりみちの公園",
  "text": "夕方、公園のベンチで ひと休みした。\n小さな勝利の あと味が する。",
  "lat": 38.26,
  "lon": 140.88,
  "reached": false
}
```
